Image Processing¶

Now that we have captured our image we can use the canny edge finder in openCV to find the edges present in the image. This snippet of code loads the image, finds the edges in the image, then defines each edge as a separate contour. Finally, the drop contour is extracted by selecting the outermost contour.
In [3]: 
img = cv2.imread('python.jpg',cv2.IMREAD_GRAYSCALE)
edges = cv2.Canny(img,100,200)

ret,mask = cv2.threshold(edges,30,255,cv2.THRESH_BINARY_INV)
mask = np.ma.masked_where(mask == 255, mask)

contours,hierarchy = cv2.findContours(edges, cv2.RETR_TREE, cv2.CHAIN_APPROX_NONE)
cnt = contours[np.max(hierarchy)]

plt.subplot(221),plt.imshow(img,cmap='gray')
plt.title('Original image'), plt.xticks([]), plt.yticks([])
plt.subplot(222),plt.imshow(mask,cmap='gray')
plt.title('Edges extracted by Canny edge finder'), plt.xticks([]), plt.yticks([])
plt.subplot(223),plt.plot(cnt[:,0,0],cnt[:,0,1],'red'),plt.imshow(mask,cmap='gray')
plt.title('Outermost extracted contour overlaid on edge image'), plt.xticks([]), plt.yticks([])
plt.subplot(224),plt.plot(cnt[:,0,0],cnt[:,0,1],'red'),plt.imshow(img,cmap='gray')
plt.title('Extracted contour overlaid on original image'), plt.xticks([]), plt.yticks([])
